# GatsbyJs Ecommerce Admin Panel

Admin Panel built using Sanity.io

[Live Demo](https://gatsbyjs-ecommerce.sanity.studio/)

## To use

- Fork or download this repository
- Ready!

## Setup

Run:

```
yarn install
```

To load some dummy data:

```
yarn import
```

## Development

To start development server

```
yarn start
```

## Deployment

```
yarn deploy
yarn deploy-graphql
```
